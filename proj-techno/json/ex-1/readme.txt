Exercice 1 : Exploration et compréhension de JSON

a. JSON (JavaScript Object Notation) constitue le standard actuel pour les échanges de données sur le Web. Il s'agit d'une syntaxe pour décrire des informations structurées sous une forme proche des objets JavaScript. JSON est un format léger d'échange de données. Facile a lire et ecrire pour des humains et est aisément analysable ou générable par des machines.
